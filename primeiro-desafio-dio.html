Surgimento da internet

A World Wide Web foi criada em 1989 pelo cientista britânico Tim Berners-Lee. A web tem dois componentes principais: clients e servers. Os clients são os programas ou dispositivos que solicitam recursos da web, como um navegador de internet, enquanto os servers são os computadores que fornecem os recursos solicitados pelos clients. Para criar aplicações web, é necessário utilizar uma linguagem de programação, como HTML, CSS, JavaScript, PHP, Ruby ou Python.

Como funiona o HTML

O HTML é uma linguagem de marcação utilizada para criar páginas web. Sua estrutura básica é composta por duas partes: o cabeçalho, que contém informações sobre a página, e o corpo, onde são colocados os elementos que formam o conteúdo da página.

Em resumo, as ferramentas do HTML incluem tags para definir o conteúdo e a estrutura da página, bem como para formatar o texto, criar listas e links para outras páginas ou recursos.

(Voltar)

O que são tags

As tags são elementos fundamentais do HTML e são utilizadas para definir o tipo de conteúdo que está sendo inserido na página. Existem tags para texto, imagens, links, listas, entre outras.

As tags de texto são utilizadas para definir o tamanho, a cor, o estilo e outras características do texto. As tags de listas são utilizadas para criar listas ordenadas e não ordenadas. Já as tags de link são utilizadas para criar links para outras páginas da web ou para outros recursos, como arquivos de áudio ou vídeo.
